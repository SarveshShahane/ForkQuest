1. BFS (Breadth First Search) -> 
Perform Breadth First Search (BFS) on a binary tree starting from the root node and print the nodes in level order.

2. DFS (Depth First Search) ->
Perform Depth First Search (DFS) on a binary tree starting from the root node and print the nodes in the order they are visited.

3. Order of Traversal ->
Perform Inorder, Preorder, and Postorder traversals on a binary tree and print the nodes in each order respectively.
