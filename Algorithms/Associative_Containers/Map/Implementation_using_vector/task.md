Implement a working Map function using vectors with following functions:

1. insert –> Takes a key–value pair and inserts or updates it in the map.
2. erase –> Takes a key and removes the corresponding key–value pair if it exists.
3. find –> Takes a key and returns true if it exists in the map, otherwise false.
4. get –> Takes a key and returns its associated value (or default/invalid if not found).
5. size –> Returns the number of key–value pairs in the map.
6. empty –> Returns true if the map is empty, otherwise false.
7. clear –> Removes all key–value pairs from the map.
8. print –> Displays all key–value pairs, optionally sorted by key.
