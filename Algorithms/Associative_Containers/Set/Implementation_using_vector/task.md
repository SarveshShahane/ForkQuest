Implement a working Set function using vectors with following functions:

1. insert –> Takes an integer and adds it to the set if it doesn’t already exist.
2. erase –> Takes an integer and removes it from the set if present.
3. find –> Takes an integer and returns true if it exists in the set, otherwise false.
4. size –> Returns the total number of unique elements in the set.
5. empty –> Returns true if the set has no elements, otherwise false.
6. clear –> Removes all elements from the set.
7. print –> Displays all elements of the set in sorted order.
