Implement a working Stack using Queue with following functions:

1. push → Takes an integer and pushes it onto the stack.
2. pop → Removes the top element from the stack.
3. top → Returns the top element without removing it.
4. empty → Returns true if the stack is empty, otherwise false.
5. size → Returns the number of elements in the stack.
6. display → Displays all elements of the stack from top to bottom.
