Implement a working Queue using Stack with following functions:

1. enqueue → Takes an integer and adds it to the queue.
2. dequeue → Removes and returns the front element from the queue.
3. front → Returns the front element without removing it.
4. empty → Returns true if the queue is empty, otherwise false.
5. size → Returns the number of elements in the queue.
6. display → Displays all elements in the queue from front to rear.
